# 🚦 Smart Traffic Fine Management System

A Smart Traffic Fine Management System developed using **C++** and **MySQL** to manage traffic violations, calculate fines, process payments, and generate reports. This project was developed as part of a university coursework to demonstrate database integration and object-oriented programming concepts.

---

## 📌 Features

-  User Registration & Login
-  Role-Based Access Control
  - Administrator
  - Traffic Officer
  - Vehicle Driver
-  Traffic Violation Management
-  Automatic Fine Calculation
-  Fine Payment Management
-  Report Generation
-  MySQL Database Integration

4. Configure the MySQL connection.
5. Build and run the project.
